# UserPostsPagerSample

Test app at the interview for 'junior' android developer
 
## <a name="Main tasks"></a>Main tasks  
*   Parse JSON
*   Save data with SQLite
*   Write logs to file.
*   Intents to open side applications.
*   Represent data with ViewPager.
*   Simple animation.
